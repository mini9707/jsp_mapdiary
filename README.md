## 주요 기술 스택

- **Backend**: Java8, Spring Framework, MyBatis, eGovFramework
- **Database**: PostgreSQL, PostGIS
- **Frontend**: JSP, JavaScript, jQuery, OpenLayers, BootStrap
- **API 통합**: GeoServer, ITS 국가교통정보센터 (실시간 교통정보, CCTV)
- **ETC**: 전자정부 표준프레임워크 기반
