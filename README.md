# MapDiary

**MapDiary**는 사용자가 자신의 위치를 저장하고, 공유하며, 다른 사용자와 상호작용할 수 있는 위치 기반 서비스입니다.

---

## 프로젝트 개요

- **프로젝트명**: MapDiary  
- **주요 기능**:  
  1. 사용자는 자신의 위치를 저장하고 관리할 수 있습니다.  
  2. 위치를 다른 사용자와 공유하고 좋아요를 받을 수 있습니다.  
  3. 사용자 간 상호작용을 통해 위치 기반 커뮤니티를 형성합니다.
